Getting Started

DiagrammeR can be installed as an R package in one of two ways: through GitHub using the devtools package, or, through CRAN.

GitHub

This is the version in development. It's the most up-to-date version, certainly, but it may not be as stable as the CRAN version.

install_github('rich-iannone/DiagrammeR')

CRAN

The stable release. While this is likely to be more stable than GitHub version, CRAN releases are far less frequent.

install.packages('DiagrammeR')
