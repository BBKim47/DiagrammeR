The R Package Known As DiagrammeR

DiagrammeR allows you to generate flowchart diagrams using text in a Markdown-like syntax. The package leverages the power of htmlwidgets to bind JavaScript with R. This intertwining of the R + JS can allow us to do a lot of really great things.

Simple (Yet Powerful) Syntax

The syntax (either mermaid or Graphviz DOT) is easy to pick up. Templating features allow for easy integration with R code.

Syntax Coloring and Previewing

While editing mermaid (.mmd) files or Graphviz (.gv) files inside RStudio, get helpful syntax coloring and easily preview results.

Scalable Vector Graphics

Beautiful output. Diagram elements such as nodes and edges are highly customizable, and, they are rendered as SVG graphics.

Versions

  • 0.1

    The First Release

    It's an htmlwidget for R. Create mermaid graphs. Use the shiny app to create graph specifications interactively.

  • 0.2

    Graphviz is added

    Graphviz has quite the legacy for creating quality graphs. Now it's in DiagrammeR. Create graph specifications in the DOT language and take advantage of mixing R into it.

  • 0.3

    More Graphviz Engines

    Change the way your diagram is rendered by taking advantage of all of these Graphviz engines: dot, neato, twopi, and circo.

  • 0.4

    Templating for Graphviz

    An easier way to inject some R into your Graphviz graph specification. The @@ marks the spot for an evaluated R expression residing in the footnotes section.
